# Assets Folder

Place your payment QR codes here:

1. **`wechat-qr.png`** - Your WeChat payment QR code (微信支付二维码)
2. **`alipay-qr.png`** - Your Alipay payment QR code (支付宝二维码)

## Recommended Image Size

- Width: 200-300 pixels
- Format: PNG (transparent background preferred)
- Square aspect ratio works best

## How to Get QR Codes

### WeChat (微信)
1. Open WeChat → Me → Services → Money → Receive Money
2. Screenshot and crop the QR code

### Alipay (支付宝)
1. Open Alipay → Receive → Save QR Code
2. Crop the QR code image
